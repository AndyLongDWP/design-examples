## Contributing

Add a new folder with it's own ReadMe file, giving clear instructions on how people can re-use your code.

If possible, supply the following to help distinguish if this could be a pattern:
- visual examples
- name
- description
- services using it

Notes on:
- design
- user reserch
- development
- accessibility
